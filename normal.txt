There is an executable under the "Scripts" directory in the virtual environment, called "activate",
running that script activates the virtual environment.
If within the "Code" directory, which is where the workspace is situated for this project,
then the virtual environment can be activated by:
"Bachelor_venv\Scripts\activate"
